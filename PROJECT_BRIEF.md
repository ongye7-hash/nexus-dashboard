# Nexus Dashboard - Project Brief

> 이 문서는 프로젝트의 현재 상태, 방향성, 미래 가치에 대해 다양한 AI의 의견을 구하기 위해 작성되었습니다.

---

## 1. 프로젝트 개요

### 한 줄 요약
**Nexus Dashboard**는 로컬 컴퓨터의 모든 개발 프로젝트를 자동으로 감지하고, 한 곳에서 관리할 수 있는 웹 기반 대시보드입니다.

### 탄생 배경
- 개발자의 바탕화면/폴더에는 수십~수백 개의 프로젝트가 흩어져 있음
- "저번에 만든 그 프로젝트 어디 있지?" 하는 상황이 반복됨
- 프로젝트마다 VSCode 열고, 터미널 열고, 서버 키는 과정이 번거로움
- 어떤 프로젝트가 배포됐는지, 마지막으로 언제 작업했는지 파악이 어려움

### 핵심 가치 제안
1. **자동 감지**: 폴더만 지정하면 Next.js, React, Python 등 프로젝트 자동 인식
2. **원클릭 실행**: VSCode, 터미널, 서버 실행을 클릭 한 번으로
3. **시각적 관리**: 카드/리스트 뷰로 모든 프로젝트를 한눈에
4. **키보드 중심**: Linear 스타일의 빠른 단축키 기반 UX

## 5. 잠재적 방향성 분석

### 방향 A: 포트폴리오 자동 생성기

**컨셉**: 프로젝트 선택 → 자동으로 포트폴리오 웹사이트 생성 → 원클릭 배포

**장점**:
- 명확한 수요 (취준생, 이직러, 프리랜서)
- 이미 프로젝트 데이터가 구축되어 있어 자동화 용이
- 차별점: 기존 포트폴리오 빌더는 수동 입력 필요
- Product Hunt 등에 론칭하기 좋은 형태

**단점**:
- 스크린샷 자동 생성 등 기술적 난이도
- 디자인 템플릿 다양성 필요
- 유지보수/호스팅 부담

**예상 타겟**: 취업 준비 개발자, 이직러, 프리랜서
**수익화**: 프리미엄 템플릿, 커스텀 도메인 연결

---

### 방향 B: 프리랜서/에이전시 관리 도구

**컨셉**: 프로젝트 + 클라이언트 + 비용 + 수익을 한 곳에서 관리

**장점**:
- 실제 돈과 연결된 문제 해결 (도메인 갱신 잊어서 사이트 다운 등)
- 기존 도구들(Notion, Excel)은 수동 관리 필요
- 반복 수익 모델 가능 (SaaS)

**단점**:
- 회계/인보이스는 법적 요구사항이 복잡
- 결제 연동 필요
- 경쟁자가 많음 (FreshBooks, Wave 등)

**예상 타겟**: 프리랜서 개발자, 소규모 에이전시
**수익화**: 월정액 구독, 프로젝트 수 제한

---

### 방향 C: AI 코드 허브

**컨셉**: 내 모든 프로젝트를 AI가 이해하고, 자연어로 검색/분석

**장점**:
- "내가 예전에 짠 결제 코드 어디있지?" 문제 해결
- 코드 재사용 촉진 (생산성 향상)
- AI 트렌드와 부합
- 기술적으로 차별화 가능

**단점**:
- LLM API 비용 (OpenAI, Claude)
- 로컬 LLM은 성능 한계
- 코드 프라이버시 우려

**예상 타겟**: 프로젝트가 많은 시니어 개발자
**수익화**: API 호출 기반 과금, 프리미엄 분석

---

### 방향 D: 로컬 퍼스트 개발자 허브

**컨셉**: GitHub/클라우드 의존 없이 완전 로컬에서 동작하는 개발 관리 도구

**장점**:
- 프라이버시 중시하는 개발자 증가
- 인터넷 없이 동작
- 기업 환경 (보안 정책으로 클라우드 금지)에서 사용 가능

**단점**:
- 동기화/백업 문제
- 협업 기능 한계
- 마케팅 어려움

**예상 타겟**: 프라이버시 중시 개발자, 기업 내부 사용
**수익화**: 기업 라이선스, 데스크톱 앱

## 6. 현재 개발자(Claude)의 의견

### 추천 방향: 포트폴리오 자동 생성기 (방향 A)

**이유**:

1. **기존 자산 활용도 최대**: 이미 프로젝트 스캔, 기술스택 추출, README 파싱 등이 구현되어 있음. 포트폴리오 생성은 이 데이터를 재가공하는 것.

2. **명확한 Pain Point**:
   - 개발자 취업 시장에서 포트폴리오는 필수
   - 하지만 대부분 "나중에 만들어야지" 하다가 미룸
   - 이유: 귀찮음, 시간 없음, 디자인 못함

3. **경쟁 차별점**:
   - 기존 포트폴리오 빌더: 처음부터 수동 입력
   - Nexus: 이미 있는 프로젝트에서 자동 추출
   - "5분 만에 포트폴리오 완성" 가능

4. **확장성**:
   - 포트폴리오 → 이력서 자동 생성
   - 포트폴리오 → LinkedIn 프로필 연동
   - 포트폴리오 → 프리랜서 프로필 (긱 이코노미)

5. **수익화 용이**:
   - 무료: 기본 템플릿 1개
   - 유료: 프리미엄 템플릿, 커스텀 도메인, AI 설명 생성

### 권장 MVP 기능

1. 프로젝트 선택 UI (체크박스)
2. 템플릿 선택 (2-3개)
3. 자동 생성된 포트폴리오 미리보기
4. HTML/정적 파일 내보내기
5. Vercel 원클릭 배포
