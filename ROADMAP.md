## Tech Stack

- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Icons**: Lucide React
- **Command**: cmdk
- **Language**: TypeScript

---

## Design Philosophy

- **UX**: Linear - 미친 속도, 단축키 기반
- **UI**: Stripe - SaaS 비주얼의 정석
- **Product**: Figma - 도구 자체가 예술
- **Brand**: Notion - 독보적 분위기
